# OPD Token Allocation Engine

Backend assignment for Medoc Health IT Pvt Ltd.

## Overview
This project implements a token allocation system for a hospital Outpatient Department (OPD). It manages doctor schedules, enforces hard capacity limits on time slots, and dynamically allocates tokens based on patient priority.

## Features
- **Fixed Time Slots**: Hard capacity enforcement.
- **Prioritization**: Supports Emergency, VIP, Follow-up, and Walk-in patients.
- **Dynamic Reallocation**: Handles cancellations and emergency insertions.
- **Simulation**: Includes a simulation script to demonstrate one OPD day.
